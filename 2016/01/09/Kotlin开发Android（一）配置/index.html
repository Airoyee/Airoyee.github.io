Kotlin开发Android（一）配置

  • Kotlin

​ 本系列记录用kotlin开发一个小app的过程，并在其中讲解一些kotlin的语法和Kotlin开发Android的优势。有关kotlin的优点和缺点不再赘述。

新建工程

​ 我写的是一个关于囚徒健身的辅助app，所以工程名为MuscleFriend的普通Android工程。报名随便起我的叫com.nuttysoft.musclefriend。

环境搭建

​ 妈蛋，又是环境搭建！说环境搭建就是扯犊子而已，就是在As上装两个插件的事，而且现在的最新版本的As是自带buff出场的，本身就已经集成了kotlin和Kotlin Extensions For Android,只需要配置gradle文件就可以了。

​ 但是, kotlin 和 Kotlin Extensions For Android插件也是有用处的。装完插件后，在As的Code菜单中灰多出来Convert Java File To Kotlin选项，用这个就可以吧普通的java文件转为kotlin文件了，神器～在Tools菜单中也会出现kotlin配置相关的选项。

​ 综上，装不装插件你看着办，装插件如下：

​ Setting－>Plugin->Install JetBrains plugin->搜索kotlin－>选中kotlin在右侧待你install安装－>选中Kotlin Extensions For Android安装－>重启，就成功了。

由于网络原因安装失败也不要担心，不耽误咱写代码。
